# ⚜️ MINUA Atelier (미누아 아틀리에)

> **우아함과 정교함 속에 피어나는 프리미엄 하이엔드 주얼리 & 레저 액세서리 셀렉션**
> 
> MINUA Atelier는 최고급 실버 주얼리, 가죽 키링, 귀걸이, 목걸이 등 엄선된 컬렉션을 다국어(한국어, 영어, 일본어) 및 실시간 동기화 데이터베이스 구조로 서비스하는 고품격 하이엔드 주얼리 쇼핑몰 아틀리에 플랫폼입니다.

---

## 🎨 주요 미학 및 디자인 철학 (Design Philosophy)

*   **Swiss-Modern & Editorial Design**: 폰트 페어링("Inter" Sans-serif & "Playfair Display" Serif) 및 넓은 여백 사용, 정밀하고 깊이 있는 차콜 톤과 미색(Off-white) 베이스의 디자인입니다.
*   **고화질 비주얼 아이덴티티**: 아래의 엄선된 이미지 에셋을 통해 예술적 무드와 정제된 인터페이스의 완벽한 조화를 지향합니다.

## 💎 핵심 구현 기술 및 아키텍처 (Tech Stack & Core Features)

1.  **Frontend**: React (Vite) + Tailwind CSS + Lucide Icons + Motion (for Smooth Route Transitions)
2.  **Persistence Backend**: **Firebase Firestore DB** 와 **Firebase Storage**
3.  **관리자 권한 제어**:
    *   일반 사용자는 깨끗한 보기 전용 카테고리 필터와 아티스틱 홈 화면만 이용 가능.
    *   어드민 계정으로 로그인 시 상단 헤더에 **관리자 대시보드** 인입 버튼 활성화.
4.  **카테고리 대표 이미지 관리 통합 설계**:
    *   **반지, 팔찌, 키링, 귀걸이, 목걸이, 선물** 총 6개 카테고리 관리 시스템 연동.
    *   관리자가 대표 이미지를 교체하면 파일 선택창을 통해 이미지가 압축 처리된 후, **Firebase Storage downloadURL로 다이렉트 업로드**됩니다.
    *   업로드 이후 즉시 Firestore의 `/categories/*` 컬렉션 문서 구조(`categoryImageUrl`)에 영구 동기화되어, 페이지 새로고침을 거쳐도 견고하게 유지됩니다.
    *   기본 디자인으로 간편하게 되돌릴 수 있는 **"기본 디자인으로 원복"** 버튼을 일괄 지원하여 편의성을 확보했습니다.
